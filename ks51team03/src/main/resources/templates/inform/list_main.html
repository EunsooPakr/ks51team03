담당자 기능
송재익(팀장)
  • 메인화면 (index)
  • 프로젝트 역할
    • 주 1회 이상 온라인-오프라인 회의 진행 주도
    • 담당 기능 URI 매핑, 기능 정의, DB Process
    • MVC 패턴에 맞춘 아키텍쳐 설계
    • 중간, 완료 보고서 및 Excel 통합 구현 등 문서 작업 총괄 진행
  • 장례 서비스 예약
    • 예약 바로 가기[ 로그인 후 장례 업체 서비스 선택 필수 ]
    • Stream API를 활용한 서비스 코드별 이미지 그룹화
    • @RequestParam, @RequestBody를 활용한 업체별 동적 서비스화
    • Thymeleaf 템플릿 엔진을 사용하여 동적 콘텐츠를 생성하고 렌더링
    • jQuery를 사용하여 DOM 조작 및 이벤트 처리
    • Axios를 사용하여 비동기 요청을 보내고 서버와 데이터 교환
  • 장례 서비스 결제
    • 결제 바로 가기[ 예약 후 결제 가능 ]
    • Payment Gateway API를 사용하여 결제 처리
    • 결제 완료 후 콜백을 처리하기 위한 WebHook 구현
    • JSON 형식의 데이터를 파싱하고 비즈니스 로직을 처리
    • 업체 및 사용자별 결제 내역을 관리하고 조회하는 기능 제공

  • 작업물
    • 1. 지도
      • 카카오 api이용 후 지도 출력
      • 키워드에 맞는 업체 리스트 검색
      • 검색된 데이터 현재 위치에 맞게 나열
      • 원하는 업체 클릭시 업체 정보 확인
      • 업체 이름 클릭시 업체 상세 정보 및 리뷰 & 문의
      • 서비스 계약을 한 업체는 특정 키워드 검색시 우선 냐열(동물병원, 동물약국, 동물장례)
    • 2. 업체관련
      • 업체 정보 페이지 확인 및 수정 삭제(업체 직원)
      • 업체 리뷰 리스트 확인 및 삭제(업체 직원)
      • 업체 문의 리스트 확인 및 답변 삭제(업체 직원)
      • 업체 구독자에게 알림 발송 기능(업체 직원)
      • 업체 직원 승인 페이지(업체 대표)
      • 업체 직원 신청 페이지(일반 회원)
    • 3. 마이페이지(회원관련)
      • 문의 작성(일반 회원)
      • 리뷰 작성(일반 회원)
      • 업체 정보 페이지 확인(일반 회원)
      • 나의 문의 및 리뷰 확인 및 수정 삭제 페이지(일반 회원)
      • 업체 구독(일반 회원)
    • 4. 파일
      • 파일(이미지) 삽입 작업
